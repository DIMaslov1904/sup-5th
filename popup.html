  1. Переходим в нашу таблицу доступов
  2. Проверяем название листа. Должно быть так же "Доступы к сайтам"
  3. Проверяем столбцы, добавляем для удобства столбец админка
    • Название
    • Адрес
    • Админка
    • Логин
    • Пароль
  4. В верхнем меню выбираем Расширения -> Apps script
  5. Копируем код:
  6. Вставляем код в редактор, удаляя пустую функцию по умолчанию
  7. Нажимаем сверху кнопку Начать развертывание
  8. Новое развертывание
  9. Выберите тип, нажимаем шестеренку, выбираем Веб-приложение
  10. У кого есть доступ, ставим - Все
  11. Предоставить доступ, дальше авторизация через гугл
  12. На странице с восклицательным знаком - Advanced -> Go to [название_проекта] (unsafe)
  13. Копируем URL
  14. Вставляем в поле API URL в разделе настроек
